Zamboni REST API

The Zamboni API provides read-only access to raw game data, player statistics, and combined reports. All endpoints return JSON and are public (no auth required).

GET /api/raw/games

Returns all active and finished games with metadata. 

            [
              {
                "game_id": 21,
                "fnsh": false,
                "gtyp": 1,
                "venue": 23,
                "time": 295,
                "sku": 1000,

                "shootout": 0,
                "pnum": 3,
                "plen": 1200,
                "ot": 0,
                "created_at": "2025-10-19T02:06:41.023Z"
              }
            ]

GET /api/players

Returns a list of all known player gamertags. 

          [
            "Kaap0",
            "Kaupoi",
            "KayBoi",
            "0paaK"
          ]

Returns full player stats and history by gamertag.
